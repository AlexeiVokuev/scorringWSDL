scorringWSDL

WSDL сервис, который рассчитывает количество баллов по входящей анкете пользователя для одобрения или отказа
в получении кредита. Анкеты пользователей и вычисленные результаты сохраняются в БД

Требуется:
Oracle XE 11g
Скрипт для создания схемы БД - export.sql

В ScorringServicePub хардкодный url
localhost должен быть определен (и единственный)

http://localhost:1986/wss/hello?wsdl
